Explore Page

SenTweets is a project created to show the sentiment or the emotion of the public towards using tweets on War on Drugs from year 2016 to 2020. Let's explore on how can we use SenTweets!

Login & Registration Page

The highlighted part on the top right of the page is where we can locate the login and registration page of SenTweets.
Once clicked it will show the login page, where the user can login the credentials. If they don't have any credentials they can click the Sign Up button to create an account.
After the Sign Up button has been clicked, they can fill in their details and they will directed to the homepage of SenTweets.

Homepage

User will be directed to the homepage for default, and it is consisted of three parts.

Browse Topics

The highlighted part shows the topics that are available on SenTweets, the users can click topics or click more to see all available topics.
Users can select the topics available and after selecting a topic it will show all discussion on the Topic Discussions on the homepage.

Topic Discussions

After selecting a topic it will show all topic discussions available from the topic, and users can select a discussion from the topic. Users can just select the discussion name and it will be directed to the discussion page.
Users can see the discussion for the given topic and they can also comment below to give feedback to the developers and interact with other users.

Recent Activities

On the highlighted part of the page. It illustrates user interactions to each topic discussions, users interaction will be registered to the Recent Activities part of the page.
